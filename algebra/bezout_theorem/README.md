# Bezout Theorem
## Definition
![Bezout Theorem definition](definition.png)

#### LaTex code
```latex
a, b \in \mathbb{Z} \wedge a,b \neq 0 \\
\exists r,s \in \mathbb{Z}\text{:} \\
gcd(a, b) = ar + bs
```
